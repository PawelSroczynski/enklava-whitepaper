## Value Appreciation Mechanics

### Supply Reduction

```
┌─────────────────────────────────────────────┐
│         DEFLATION MECHANICS                  │
├─────────────────────────────────────────────┤
│                                              │
│   Hospitality Revenue ──► 20% to Burns      │
│   Hardware Sales ──────► 2.5% to Buybacks   │
│   Access Events ───────► Burn-to-Book       │
│                                              │
│   Result: Circulating supply decreases      │
│           while utility demand increases    │
│                                              │
└─────────────────────────────────────────────┘
```

### Value Accretion Formula

```
Token Value = f(
    Revenue Growth,        // More buybacks
    Supply Reduction,      // Burns + staking
    Ecosystem Expansion,   // Franchise network
    Utility Demand         // Access requirements
)
```

## Terminal Backstop

Even in worst-case dissolution:

1. **Foundation Mandate** requires physical asset liquidation
2. **Terminal Buyback** executed for all token holders
3. Land value (€160,000+) provides floor

**Floor Value Estimate:** €160,000 ÷ 10,000,000 = **€0.016/token minimum**

This represents a 5x downside protection at seed price (€0.08).
