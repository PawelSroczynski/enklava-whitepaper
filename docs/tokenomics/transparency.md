# Transparency & Adoption

Trust is built through radical transparency. This section details the public infrastructure for verifying protocol health and the user-friendly onboarding path for non-crypto-native participants.

---

## Bio-Dashboard (stats.enklava.com)

A real-time public dashboard displaying all key metrics—no login required, no trust required.

|----------|--------|------------------|

### Verification Links

Every metric links to its source:

| Metric | Verification |
|--------|--------------|
| Token transactions | Polygonscan link |
| Buyback executions | Multisig transaction history |
| Revenue claims | Quarterly audited reports |
| Physical metrics | IoT sensor data (Digital Twin) |

### Why This Matters

Traditional investments require trusting PDF reports. EKLV holders can:

1. **Verify buybacks** — Every purchase visible on-chain
2. **Calculate P/E ratio** — Revenue metrics vs. market cap
3. **Track burn rate** — Supply reduction over time
4. **Monitor operations** — Real infrastructure, real data

## User Adoption Protocol: The "Managed Wallet"

### The Problem

Web3 onboarding friction kills conversion:

| Barrier | Drop-off Rate (Industry Avg) |
|---------|------------------------------|
| "Install a wallet" | 60% abandon |
| "Write down 12 words" | 40% abandon |
| "Buy ETH for gas" | 30% abandon |
| "Bridge to Polygon" | 25% abandon |
| **Cumulative** | **~90% never complete** |

### The Solution: Custodial-First Architecture

Users don't need to understand crypto to participate. They earn value; crypto details come later.

---

### Stage 1: Managed (Months 1-12)

| Aspect | Implementation |
|--------|----------------|
| **User Experience** | Users see "Credits" in their Toloka dashboard |
| **Backend** | Company holds private keys in secure custodial infrastructure |
| **Value Source** | Toloka contributions, referrals, purchases |
| **Withdrawal** | Disabled — "Coming Soon" message |
| **User Knowledge Required** | Zero crypto knowledge |

**What Users See:**
```
┌─────────────────────────────────────┐
│  Your Balance: 1,250 Credits        │
│  ─────────────────────────────────  │
│  Earned this month: +150            │
│  Lifetime earned: 1,250             │
│                                     │
│  [Withdraw] Coming Soon             │
└─────────────────────────────────────┘
```

**Rationale:** Users engage with the ecosystem without friction. Value accumulates transparently. They learn about EKLV naturally over time.

---

### Stage 2: Optional Early Access (Months 6-12)

| Aspect | Implementation |
|--------|----------------|
| **Eligibility** | Users who complete KYC verification |
| **Action** | Can request early withdrawal to personal wallet |
| **Fee** | 2% withdrawal fee (covers gas + admin) |
| **Limit** | Maximum 50% of balance per month |
| **User Knowledge Required** | Basic wallet setup |

**What Users See:**
```
┌─────────────────────────────────────┐
│  Your Balance: 2,500 Credits        │
│  ─────────────────────────────────  │
│  Ready to withdraw? Complete KYC    │
│  to unlock early access.            │
│                                     │
│  [Complete KYC] [Learn About EKLV]  │
└─────────────────────────────────────┘
```

**Educational Flow:**
1. User completes KYC
2. Tutorial: "What is EKLV?"
3. Tutorial: "Setting up a wallet"
4. Tutorial: "Receiving your tokens"
5. Withdrawal enabled

## Custodial Risk Mitigation

Holding user funds creates responsibility. Here's how we manage it:

| Risk | Mitigation |
|------|------------|
| **Company Failure** | Credits convertible to EKLV claim; Foundation executes Terminal Backstop |
| **Hack/Theft** | Multi-sig custody; insurance coverage (target: €500K) |
| **Regulatory Action** | Legal structure separates custodial ops from Foundation |
| **Key Loss** | 2-of-3 multisig ensures no single point of failure |
| **Employee Malfeasance** | Separation of duties; audit trail on all operations |

### Insurance Coverage

| Coverage Type | Target Amount | Status |
|---------------|---------------|--------|
| Crypto custody insurance | €500,000 | In progress |
| Directors & Officers | €1,000,000 | In progress |
| Cyber liability | €500,000 | In progress |
